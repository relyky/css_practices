# Sample
### Example: 用`CSS Grid`取代`<table>`語法
若只是設計靜態表格，用傳統的`<table><tr><td>...` 的語法仍可以滿足。   
然而現在都要求表格要支援 RWD 效果，這時建議用`CSS Grid`這個新設計的CSS功能語法會好過一些。  
當然`CSS Grid`除了可以取代`<table>`還可以做很多事。
